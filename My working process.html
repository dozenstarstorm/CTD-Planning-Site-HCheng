The strengths of using fireworks

  • 1.You can work on more than one image in more than one page at a time.
  • 2. It makes slicing and exporting small parts of a compostion simple.
  • 3.You can build quick and easy vectors.
  • 4. You can easily plan and build a splitting image.
  • 5. You can save a lot of money.

The problems of using fireworks

  • 1.You must export to save file in format other than .png
  • 2.Toolbox is not intuitive.
  • 3.It has limited filters, layer styles and effects.
  • 4. It does not have sub category support in the layers menu.
  • 5. There is no "browse" images feature.

The strengths of using dreamweaver

  • 1.You can quickly spot any errors from a cluster of codes (e.g. I can easily see a missing closing tag of "/html" to avoid the pages not showing up.)
  • 2.You can use the property bar to create links.
  • 3.You can create bigger and more manageable sites.
  • 4.You can speed up your development time by seeing quickly what files are open and edit connected files.

The weaknesses of using dreamweaver

  • 1.Advanced functionality can be confusing to users, a lot of interface panels require steep learning and understanding in order to acquire the skills and execute.
  • 2.It is very expensive, maybe not worth the money if not for frequent uses.
